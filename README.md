# 🚀 Skill Learning Platform (Frontend)

A modern and responsive learning platform UI built using **React + Tailwind CSS**.
This project showcases a clean, interactive interface for browsing courses, filtering by age/time, and exploring categories.

---

## 📸 Features

* 🎯 Hero section with search bar
* 📚 New Launches & Featured Courses (horizontal sliders)
* 👶 Age-based filtering system
* ⭐ Ratings & course cards UI
* 👨‍🏫 Top Teachers section
* 🧠 Popular Categories (interactive with animations)
* ⏰ Time-based filtering (with highlight & selection)
* 🧭 Sticky Navbar with smart fade effect
* 🎨 Fully responsive & modern UI

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 🧩 React Icons
* 💡 JavaScript (ES6+)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/KunalGangarde/UndoSchools.git
cd UndoSchools
```

---

### 2️⃣ Install dependencies

```
npm install
```

---

### 3️⃣ Run the project

```
npm run dev
```

👉 Open: http://localhost:5173
